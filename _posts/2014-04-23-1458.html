検索したらすぐ見つかった

HarooPadデフォルトの日本語フォント変だけど、CSSにfont-family書けばちゃんとフォント変更できるよ— しょーへー90.9 (@shohei909) 2014, 3月 27

ということで早速、File->Preference->Viewer->Your Style->Edit して
editor/default.cssに
css
editor {
font-family: 'Lucida Grande','Hiragino Kaku Gothic ProN', 'ヒラギノ角ゴ ProN W3',Meiryo,メイリオ,sans-serif;
}
viewer/default.cssに
css
*{
font-family: 'Lucida Grande','Hiragino Kaku Gothic ProN', 'ヒラギノ角ゴ ProN W3',Meiryo,メイリオ,sans-serif;
}
な感じで追記して、上書き保存。バッチリ。
あとアイコン代えれればいいかんじかな。
