ささっとSafariで確認ながら組んだページをWinIEで見たときの悲しさはMacでweb制作をしている人は誰もが一度は体験しますよね。
まだまだブラウザによって差があるCSSのレンダリングをなるべく均等にするcssです。（自分用）

clearfixとイメージの置き換えも書いてあります。text-indentを使ったイメージの置き換えはSPAMとされるという議論は多々あり。
僕は気にしませんが、もしコピペして使うなら、気になる人は消した方が良しです。

デフォルトのレイアウトは中央揃えです。idをwrapとしたものをbody直下に入れると、幅800pxで中央揃えのボックスができます。
<body>
<div id="wrap">.....</div>
</body>

文字と画像を置き換えるときは、文字をくくっているタグにclass="img"を追加。別途background-imageとwidth,heightをセットすれば置き換わります。
<style>
.title {
background-imege: url('title.gif');
width:200px;
height:20px;
}
</style>
<h1 class="img title">.....</h1>

* {
margin:0;
padding:0;
border:none;
}
body {
font-family:'ヒラギノ角ゴ Pro W3','Hiragino Kaku Gothic Pro','メイリオ',Meiryo,'ＭＳ Ｐゴシック',sans-serif;
font-size:0.8em;
line-height:1.6em;
text-align:center;
}
#wrap {
margin:0 auto;
text-align:left;
width:800px;
}

text-decoration:none;
}

}

}
